# 📚 Hệ Thống Kiểm Kê Sách Thư Viện

Hệ thống kiểm kê sách thư viện hoạt động hoàn toàn offline, không cần server.

## ✨ Tính năng

- 📤 Upload file Excel (.xlsx, .xls)
- 🔍 Tìm kiếm nhanh real-time
- ✍️ Nhập số lượng thực tế
- 📊 Thống kê tiến độ tự động
- 💾 Lưu dữ liệu tự động (LocalStorage)
- 📥 Xuất file Excel kết quả
- 📱 Responsive - Hỗ trợ mobile

## 🚀 Cách sử dụng

1. Mở file `index.html` trong trình duyệt
2. Upload file Excel danh sách sách
3. Tìm kiếm và nhập số lượng thực tế
4. Xuất file Excel kết quả

## 📋 Định dạng file Excel

File Excel cần có các cột:
- **Cột A:** STT
- **Cột B:** Tên sách
- **Cột C:** Số lượng hệ thống
- **Cột D:** Số lượng thực tế (tùy chọn)

## 🌐 Demo Online

Truy cập: https://lamvanna.github.io/NhapExcel/

## 📦 Cài đặt

```bash
# Clone repository
git clone https://github.com/Lamvanna/NhapExcel.git

# Mở thư mục
cd NhapExcel

# Mở file index.html trong trình duyệt
```

## 🔧 Yêu cầu

- Trình duyệt hiện đại (Chrome, Edge, Firefox, Safari)
- Không cần Internet sau khi tải trang
- Không cần cài đặt thêm

## 📝 Lưu ý

- Dữ liệu được lưu trong LocalStorage của trình duyệt
- Không upload dữ liệu lên server
- Hoạt động hoàn toàn offline
